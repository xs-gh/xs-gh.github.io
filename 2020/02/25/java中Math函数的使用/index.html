java中Math函数的使用

Math函数的调用

Math函数在java中有两种调用方法：

  1. 直接调用： Math.方法名()
  2. 导入包： import static java.lang.Math.*;
    此时调用Math函数时不需要加Math,例： 
    import static java.lang.Math.*;
public class Math01{
 System.out.println(sqrt(4));
}

当程序需要多次调用Math函数时使用第二种方法更方便。

Math常用函数

  1. Math.sqrt(): 计算平方根
    System.out.println(Math.sqrt(4)); // 2.0
  2. Math.cbrt(): 计算立方根
    System.out.println(Math.cbrt(8)); // 2.0
  3. Math.pow(x, y): 计算x的y次方
    System.out.println(Math.pow(2, 4)); // 16.0
  4. Math.max(x, y): 比较x和y的大小返回较大值
    System.out.println(Math.max(2, 4)); // 4
  5. Math.min(x, y): 比较x和y的大小返回较小值
    System.out.println(Math.min(2, 4)); // 2
  6. Math.abs(): 取绝对值
    System.out.println(Math.abs(-1.2)); // 1.2
  7. Math.round(): 四舍五入
    System.out.println(Math.round(9.98)); // 10
    【注意】当数字类型为float时，round返回int值；数字类型为double时，round返回long值。
  8. Math.roudom(): 随机取得一个[0,1)范围内的double值
    System.out.println(Math.random()); // [0,1）范围内的double值
    System.out.println(Math.sqrt()*2); // [0,2) 范围内的double值
    System.out.println(Math.sqrt()*2 + 1); // [1,3) 范围内的double值

Math三角函数

  1. Math.PI: 表示π的近似值
    System.out.println(Math.PI); // 3.141592653589793
    【补充】 Math.E 表示对数的近似值
  2. Math.sin()：正弦函数
    System.out.println(Math.sin(Math.PI/2)); // 1.0
  3. Math.cos(): 余弦函数
    System.out.println(Math.cos(0)); // 1.0
  4. Math.tan(): 正切函数
    System.out.println(Math.tan(0)); // 0.0

Math指数函数及自然对数

  1. Math.exp(): 指数函数
    System.out.println(Math.exp(2)); // 7.38905609893065
  2. Math.log(): 自然对数（以常数e为底数的对数）
    System.out.println(Math.log(1)); // 0.0
